# Manage Accounts - AI Agent Context

## Challenge Info
- **Day:** 74 of 100
- **Design:** See `design.png` in this folder
- **Starter file:** `index.html`

## User Story

As a user, I want to manage multiple accounts from a single interface, so that I can easily switch between accounts and log out of all accounts when needed.

## Acceptance Criteria

- Display the user's profile information at the top, including a profile picture and email.
- Provide a "Manage Account" button for accessing account settings.
- List all linked accounts with a clear indicator of the account type (e.g., primary or secondary).
- Include an option to add a new account to the list.
- Provide a "Log out from all accounts" button for quick access.
- Include links to the Privacy Policy and Terms of Service at the bottom.

## Technical Notes

- Use any framework or vanilla HTML/CSS/JS
- The design mockup (`design.png`) is the visual target
- Icons, fonts, and images need to be sourced (use Google Fonts, Font Awesome, Unsplash, etc.)
- Make it responsive across mobile, tablet, and desktop
- Add interactivity where it makes sense

## Suggested Approach

1. Analyze `design.png` for layout structure, colors, typography, and spacing
2. Build the HTML skeleton with semantic elements
3. Style with CSS to match the design
4. Add JavaScript for interactivity
5. Test responsiveness and accessibility
